# Package index

## All functions

- [`Aopt()`](https://chikuang.github.io/SLSEdesign/reference/Aopt.md) :
  Calculate the A-optimal design under the second-order Least squares
  estimator
- [`Dopt()`](https://chikuang.github.io/SLSEdesign/reference/Dopt.md) :
  Calculate the D-optimal design under the SLSE
- [`calc_phi()`](https://chikuang.github.io/SLSEdesign/reference/calc_phi.md)
  : Calculate the loss function of the A-, c- or D-optimal design
- [`copt()`](https://chikuang.github.io/SLSEdesign/reference/copt.md) :
  Calculate the c-optimal design under the SLSE with the given
  combination of the parameters
- [`plot_dispersion()`](https://chikuang.github.io/SLSEdesign/reference/plot_dispersion.md)
  : Verify the optimality condition for an optimal design (A-, c- or
  D-optimality)
- [`plot_weight()`](https://chikuang.github.io/SLSEdesign/reference/plot_weight.md)
  : Plot the weight distribution of the optimal design for univaraite
  regression model
